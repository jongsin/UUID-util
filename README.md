# UUID-util
UUID Utility (Shorten)

# Build
  - go build -v -o ./cmd/... gouu

# UUID To Short UUID base58.FlickrAlphabet
  - gouu -l 483ca83c-677d-11ec-949d-42010a006010

# Short UUID base58.FlickrAlphabet To UUID
  - gouu -s 9VnixksFfQgHYdE5ShW2jG
